    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your inventory system up and running in minutes
  </p>
  
  <Steps>
    <Step title="Install dependencies">
      Clone the repository and install the required Python packages.
      
      ```bash
      git clone https://github.com/mauricio-777/InventoryPython.git
      cd InventoryPython
      pip install -r requirements.txt
      ```
    </Step>
    
    <Step title="Start the backend server">
      Launch the Flask API server on port 8000.
      
      ```bash
      cd backend
      python main.py
      ```
      
      <Note>The database will be automatically created with default roles and an admin user on first startup.</Note>
    </Step>
    
    <Step title="Start the frontend">
      In a new terminal, launch the React frontend.
      
      ```bash
      cd frontend
      npm install
      npm run dev
      ```
      
      The application will be available at `http://localhost:5173`
    </Step>
    
    <Step title="Log in">
      Use the default admin credentials to access the system. The credentials are created automatically during the seed process.
      
      <Tip>Check the console output from the backend server for the default admin username and password.</Tip>
    </Step>
  </Steps>
</div>
