## Get Dashboard Summary

<Card title="GET /api/v1/reports/dashboard" icon="chart-line">
  Retrieves a comprehensive dashboard summary with all key performance indicators.
</Card>

### Query Parameters

<ParamField query="low_stock_threshold" type="integer" default="10">
  Threshold value to identify products with low stock levels
</ParamField>

### Authorization

Requires one of the following roles:
- `admin`
- `gestor`
- `consultor`

### Response

```json
{
  "status": "success",
  "data": {
    "total_inventory_value": 150000.50,
    "low_stock_count": 5,
    "recent_movements_count": 25,
    "rotation_metrics": {...}
  }
}
```

## Get Low Stock Products

<Card title="GET /api/v1/reports/dashboard/low-stock" icon="triangle-exclamation">
  Retrieves products with stock levels below the specified threshold.
</Card>

### Query Parameters

<ParamField query="threshold" type="integer" default="10">
  Minimum stock quantity threshold. Products below this value are returned.
</ParamField>

### Authorization

Requires one of the following roles:
- `admin`
- `gestor`
- `consultor`

```json

  "status": "success",
  "threshold": 10,
```
